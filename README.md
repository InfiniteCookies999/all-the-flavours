## Project Setup

#### Server
  
  * Project uses Java/Gradle and should be opened under `./server`.
  * MySQL will need to be installed and a `./server/.env` file containing
    
    ```
    DB_ADDRESS=<address>
    DB_PORT=<port>
    DB_NAME=<database name>
    DB_USERNAME=<username>
    DB_PASSWORD=<password>
    ```

### Client

  * Ensure you have NPM installed then run `npm install` to install dependencies.
  * To start the development environment run `npm start`.
